# Spatiotemporal Growing Pattern
Scripts to generate the simulation results and scMultiome analyses in Zhou et al. 2025

[The MATLAB app installer](https://github.com/Wei-BioMath/Spatiotemporal_Growing_Pattern/blob/main/Turing%20Pattern%20On%20Growing%20Domain.mlappinstall), which will generate a GUI interface to easily tune the model parameters, simulate the model and visualize the results.

[The MATLAB code](https://github.com/Wei-BioMath/Spatiotemporal_Growing_Pattern/blob/main/growing_domain_non_conserve_exponential_growth_clean.m), *.m file, used with the [function](https://github.com/Wei-BioMath/Spatiotemporal_Growing_Pattern/blob/main/mol_rhs_non_conserve.m). Running the code in MATLAB will generate simulation results of the model and figures. 

[The R code for scMultiome and CellChat analyses](https://github.com/Wei-BioMath/Spatiotemporal_Growing_Pattern/blob/main/multiome_Zhou_quail_Weighted%20Nearest%20Neighbor%20Analysis_for_publication.R)

[The R code for DIRECT-NET analysis](https://github.com/Wei-BioMath/Spatiotemporal_Growing_Pattern/blob/main/dapeak.R)
